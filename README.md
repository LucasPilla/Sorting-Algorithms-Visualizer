# Sorting-Algorithms-Visualizer

## Run

- Clone GitHub repository
```
git clone https://github.com/LucasPilla/Sorting-Algorithms-Visualizer.git
```
- Install requirements:
```
pip install -r requirements.txt
```
- Run:
```
python3 src/main.py
```
